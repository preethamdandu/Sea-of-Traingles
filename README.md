# Sea of Triangles


Control your ship in the infinite world of **Sea of Triangles**.


## Installation

## Linux


$ sudo apt-get install libglfw3-dev libassimp-dev
$ python3 -m pip install numpy glfw PyOpenGL PyOpenGL-accelerate Pillow AssimpCy


## Usage


$ cd src/
$ chmod +x main.py
$ ./main.py
